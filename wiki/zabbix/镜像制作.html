  • ENV #и®ҫзҪ®е®№еҷЁзҺҜеўғеҸҳйҮҸ
  • USER nginx #жҢҮе®ҡиҝҗиЎҢж“ҚдҪңзҡ„з”ЁжҲ·
  • RUN yum install vim unzip -y && cd /etc/nginx #жү§иЎҢshellе‘Ҫд»ӨпјҢдҪҶжҳҜдёҖе®ҡиҰҒд»ҘйқһдәӨдә’ејҸзҡ„ж–№ејҸжү§иЎҢ
  • VOLUME #е®ҡд№үvolumeпјҢжҢӮиҪҪзӮ№жң¬ең°е®ҝдё»жңә/зҪ‘з»ңи·Ҝеҫ„
