shell在线安装

#********************************************************************
#来自王晓春老师
#********************************************************************
COLOR=”echo -e \033[1;31m”
END=”\033[m”
VERSION=”19.03.5-3.el7”
wget -P /etc/yum.repos.d/ https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo || { ${COLOR}”互联网连接失败，请检查网络配置!”${END};exit; }
yum clean all
yum -y install docker-ce-$VERSION docker-ce-cli-$VERSION || { ${COLOR}”Base,Extras的yum源失败,请检查yum源配置”${END};exit; }
mkdir -p /etc/docker
cat > /etc/docker/daemon.json <<EOF
{
“registry-mirrors”: [“https://si7y70hh.mirror.aliyuncs.com"]
}
EOF

systemctl restart docker
docker version && ${COLOR}”Docker安装成功”${END} || ${COLOR}”Docker安装失败”${END

存储引擎

数据管理

发行版本 推荐存储驱动 Alternative drivers
Ubuntu overlay2 overlay¹, devicemapper², aufs³, zfs, vfs
Debian overlay2 overlay¹, devicemapper², aufs³, vfs
CentOS overlay2 overlay¹, devicemapper², zfs, vfs
SLES 15 overlay2 overlay¹, devicemapper², vfs

  • 默认通常不需要修改

  • 修改之前一定要备份数据，数据，images 会丢失

  • 修改路径：/etc/dockerdaemon.json # “Storage Driver”: “overlay2” #注意格式

  • 修改docker数据存储路径：vim /etc/dockerdaemon.json # “data-root”: “/opt/docker”

镜像加速

通过修改daemon配置文件/etc/docker/daemon.json来使用加速器。使docker仓库，指向阿里
